# Troque-agora APP

Essa é a aplicação mobile da plataforma troque-agora.

## Instalação

Use um gerenciador de pacotes como [yarn](https://yarnpkg.com) ou [npm](https://www.npmjs.com) para instalar as dependências.

```bash
yarn install
```

## Iniciando o servidor

```bash
yarn start
```
